### 获取文件大小

```cpp
#include <sys/stat.h>

// 通过stat结构体 获得文件大小，单位字节
size_t getFileSize(const char *fileName) {

    if (fileName == nullptr) {
        return 0;
    }

    // 这是一个存储文件(夹)信息的结构体，其中有文件大小和创建时间、访问时间、修改时间等
    struct stat statbuf;

    // 提供文件名字符串，获得文件属性结构体
    stat(fileName, &statbuf);

    // 获取文件大小
    size_t filesize = statbuf.st_size;

    return filesize;
}
```
